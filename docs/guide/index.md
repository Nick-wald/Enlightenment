## 安装Godot

{==在使用引擎前，请先安装`Godot`==}。

登录[Godot官网](https://godotengine.org/)，选择最新版本安装即可。

## 获取引擎文件

!!! bug

    引擎在不断更新，请在导入引擎前确认您获取的版本，以防版本冲突造成游戏崩溃。

引擎代码目前托管在GitHub和Gitee中。由于Gitee从GitHub同步有一定延迟，所以{==推荐优先获取GitHub版本==}。

=== "GitHub仓库地址"

    ```
    https://github.com/Nick-wald/Enlightenment
    ```

=== "Gitee仓库地址"

    2

下载后从`Godot`导入项目即可
