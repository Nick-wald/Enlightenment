=== "底层代码"

    !!! danger

        虽然改动底层代码可以修改程序行为，但是却极易造成游戏错误，甚至是崩溃。

        常见的错误原因包括但不限于：

        - 因自动更新造成的游戏代码版本不一致
        - 因包代码冲突造成崩溃
        - 因为游戏依赖包缺失造成引用出界

    当然，若补丁包已经无法满足需要，您也可以直接对底层代码进行改动。
